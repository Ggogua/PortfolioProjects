# 🚴 Bike Sales Dashboard - Excel Project

## Project Overview
This interactive Excel dashboard analyzes customer purchasing patterns for a bike sales dataset. It provides insights into customer demographics, income levels, and commute patterns to identify key factors influencing bike purchases.

## Dashboard Features
- **Interactive Slicers:** Filter by Marital Status, Region, Education, and Gender
- **Visualizations:**
  - **Average Income Per Purchase:** Bar chart comparing income levels of buyers vs. non-buyers by gender
  - **Customer Age Brackets:** Line chart showing purchase patterns across age groups
  - **Customer Commute:** Line chart analyzing how commute distance affects purchase decisions

## Skills Demonstrated

### Data Cleaning
- Removed duplicate records
- Standardized text values (M/F → Male/Female, M/S → Married/Single)
- Created age brackets using nested IF formulas
- Handled missing data (replaced blanks with "Unknown" where appropriate)
- Formatted income as currency
- Standardized commute distance categories

### Data Analysis
- Created Pivot Tables to summarize data
- Calculated average income by gender and purchase status
- Analyzed purchase patterns across age groups
- Examined relationship between commute distance and bike purchases

### Data Visualization
- Built interactive charts (column and line charts)
- Connected slicers to multiple Pivot Tables
- Designed a clean, professional dashboard layout
- Applied consistent color coding (Blue = Yes, Orange = No)

## Key Insights
- Middle-aged customers (31-54) are the most likely to purchase bikes
- Short commuters (0-1 miles) have the highest purchase rate
- Long commuters (10+ miles) rarely purchase bikes
- Males who purchase bikes have higher average income than females
- North America has the highest customer concentration
